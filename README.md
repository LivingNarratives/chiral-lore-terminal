# Chiral Lore Terminal

**An immersive, in-universe BRIDGES research archive**

A passionate fan-made Death Stranding lore encyclopedia that feels like discovering a classified terminal from the chiral network itself.

## Features
- Atmospheric BRIDGES terminal aesthetic
- Fully client-side (HTML/CSS/JS)
- Mobile-first responsive design
- Interactive timeline
- Searchable lore database
- Elegant modals with deep entries
- Floating chiral particles
- Theory Vault with community-style theories
- Media Gallery
- Keyboard shortcuts and subtle easter eggs

## How to View / Deploy on GitHub Pages

1. Go to your repository settings
2. Scroll to **Pages**
3. Under "Source" select **Deploy from a branch** → `main` /root
4. Save
5. Wait ~1 minute and visit the URL GitHub provides (usually `https://yourusername.github.io/chiral-lore-terminal`)

Everything works offline once loaded. No backend, no accounts, pure immersive experience.

## Philosophy
This is not a generic wiki. It is designed to feel like you hacked into a high-clearance BRIDGES terminal and are exploring forbidden knowledge about the Death Stranding.

Built with love for the universe Kojima created.
